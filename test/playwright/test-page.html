Web Extension Test Page

This page is used to test if the web extension content script is working correctly.

Test Content

If the extension is working, you should see a Vue.js component injected above this page with the ID "my-vue-header".

The extension should:

  • Create a container with ID "my-vue-header"
  • Mount a Vue.js application inside it
  • Display "Hello from App.vue ESTATE plugin"
  • Display the extension version

This is the test area where the extension should inject its content.
